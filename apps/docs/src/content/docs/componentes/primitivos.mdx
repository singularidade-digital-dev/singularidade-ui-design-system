## Categorias

### Foundations

- **Color** — paleta brand, escalas, gradient
- **Typography** — Plus Jakarta Sans + JetBrains Mono, scale, weights
- **Spacing** — escala 4px, radius, shadow
- **Motion** — durations, easings
- **Iconography** — Lucide subset + custom ss:\*
- **Logos** — todas as variantes por brand

### Vaadin core

- **Inputs** — text-field, password, integer, number, text-area, combo-box, select, date-picker, checkbox
- **Actions** — button (primary/secondary/tertiary/danger/success), menu-bar
- **Display** — avatar, badge, icon, progress-bar
- **Layout** — vertical/horizontal-layout, split-layout, tabs
- **Overlay** — dialog, notification, tooltip, popover
