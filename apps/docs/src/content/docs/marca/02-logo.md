## Símbolo

**Singularidade**: cubo isométrico 3-faces com letras S/D em negative space, gradient brand canônico (magenta → coral → orange).

**Integras**: 2 chevrons sobrepostos formando "X" estilizado, mesmo gradient.

## Variações

| Variante         | Uso                                             |
| ---------------- | ----------------------------------------------- |
| `symbol.svg`     | Avatar, favicon, social icons (sem texto)       |
| `horizontal.svg` | Header, footer, email signature                 |
| `vertical.svg`   | Hero, splash, marketing                         |
| `monochrome.svg` | Sobre fundo colorido/imagem; usa `currentColor` |

## Clear space

Espaço mínimo ao redor do logo = **1× altura do símbolo**. Não inserir nada nessa área.

## Tamanho mínimo

- **Web**: 24px (símbolo) / 96px (lockup horizontal)
- **Print/exibição grande**: 32px+ ideal pro símbolo

## Don'ts

<Aside type="caution">
- Não distorcer proporções
- Não trocar cores do gradient
- Não adicionar sombras/contornos
- Não rotacionar
</Aside>
